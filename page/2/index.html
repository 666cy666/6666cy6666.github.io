github博客搭建（三）——链接github

如果你跟上了之前的教程，那么现在应该能够在本地访问博客了，那么如何才能让其他人也能看到你的博客呢？如果想用云服务器的可以另找教程自己搭建，后续都是用github来搭，云服务器我怕到时候到期了我就懒得再去配了，github我相信应该不会跑路。

配置git密钥

按道理你们应该在第一步就已经配好了密钥，但是老规矩，上教程Git的环境配置（GitHub的SSH密钥的配置超详细）_gitshezhi github 帐户设置 ssh 密钥并运行 ssh 代理-CSDN博客

配置好密钥了之后，点击 Start project 或者下面的 new repository 建立一个新的仓库，注意Github 仅能使用一个同名仓库的代码托管一个静态站点，这里注意仓库名一定要是： 用户名.github.io。

例如我们这里是666cy666.github.io，由于我们已经创建过了，因此不能再创建了，到这一步，恭喜你，已经完成了45%！！！

上传至github

在这一步要安装一个部署插件 hexo-deployer-git，打开cmd，输如以下指令：

npm install hexo-deployer-git --save

然后修改配置和安装部署插件，根目录下的config，拉到文件末尾，填上以下配置

deploy:
  type: git
  repo: https://github.com/666cy666/666cy666.github.io.git
  branch: master

这里repo修改为你的github.io的仓库地址，注意后面是带.git的，用于推送，后面那个branch有人说是main，我这里写的master也没问题，大家根据自己的仓库改。

最后执行如下两条命令就能够部署上传啦。

# generate，生成博客
hexo g
# deploy，部署到Github上
hexo d

出现Deploy done的时候说明已经可以通过用户名.github.io访问了，恭喜你，已经完成了50%！！！

这里由于我已经绑定了域名，所以说就不演示了，事实上到这里博客基本功能已经全部实现了，快夸一夸你自己把，后续的内容都是一些让自己更加舒服的写博客，以及让别人更加舒服的看博客的内容。

github博客搭建（二）—— 博客初始化与个性化设置

hexo下载与初始化

以上环境准备好了就可使用 npm 开始安装 Hexo 了，在命令行输入执行如下命令：

npm install -g hexo-cli

安装 Hexo 完成后，在指定文件夹下打开“Git Bash”——其实这里可以不用gitbash，用cmd是一样的，gitbash后面都可以用cmd代替。

再执行下列命令，Hexo 将会在指定文件夹中初始化所需要的文件

#在"文件夹名字"初始化hexo
hexo init 文件夹名字
#切换到你创建的文件夹下
cd 文件夹名字
#npm初始化
npm install

这里如果初始化慢的话，可以去hexojs/hexo-starter: Hexo Starter site (use npx hexo init myBlog) (github.com)下载zip解压到文件夹下，然后再npm install

解压后的文件，之后在当前文件夹下npm install

npm install初始化后就可以尝试启动了

hexo基本指令

hexo g(generate)

这个命令会根据你的源文件（如 Markdown 文件）生成静态 HTML 文件，并将它们放在 public 目录中。每次你更新文章或修改配置文件后，都需要运行这个命令来重新生成静态文件

hexo s(server)

这个命令会启动一个本地服务器，通常监听在 http://localhost:4000。你可以在本地预览你的博客，而无需先生成静态文件。这个命令还会监听文件变化，并自动重新加载页面，方便开发调试。

hexo d(depoly)

这个命令会根据你在 _config.yml 中的配置，将生成的静态文件部署到远程服务器上。常用的部署目标包括 GitHub Pages、FTP 服务器等。你需要先配置好部署选项，然后运行这个命令将网站发布到指定位置。

hexo new page 标签名字

这个命令会在 source 目录下新建一个独立页面目录，并生成相应的 index.md 文件。例如，运行 hexo new page about 会在 source 目录下创建一个 about 文件夹，里面有一个 index.md 文件，你可以在其中编写“关于”页面的内容。

尝试运行

hexo g
hexo s

执行完上面的指令后，若能成功在本地:4000端口访问，恭喜你，已经成功11.4514%了！！！
