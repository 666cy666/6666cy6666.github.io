github博客搭建（一）—— 环境配置

大一的时候在csdn上写博客，后面发现就我那写的小教程竟然也有人盗，后面就把账号注销了，虽然说csdn上好教程确实不少，但是史里淘金的过程真得很难评，痛定思痛，大二大三就一直在本地写markdown文档了，现在大三暑假我的电脑终于撑不住了，正好准备重装系统，干脆把之前踩坑写的md整理一下搞个博客算了。

原先准备是在云服务器上搭个动态博客的，后面想想动态的无非多了个统计还有动态修改功能，我个人不是很在意这些，反正在本地写已经习惯了，大不了修改了再上传呗，多大点事。

于是从头跟着b站还有一些网站教学搞了一个下午终于搞定了，搞完之后真的成就满满，那些教程有些都是几年前的了，虽然说避免了很多坑，但是也踩了不少新坑，感觉还是有必要写一下博客的。

环境准备

  • Node.js：JavaScript 运行时环境，用于构建和运行博客相关的应用程序。
  • Git：版本控制工具，用来上传和管理博客文件。
  • Typora：Markdown 编辑器，用于书写和编辑博客内容。
  • 一个域名：为你的博客提供一个独立的访问地址。
  • Python 环境：用于批量处理 Markdown 图片格式。如果没有 Python 环境，后续的步骤中我会提供一个可执行文件（exe），但不同主题模板可能需要自行调试。
  • CDN 加速（Cloudflare）：由于 GitHub 的服务器位于国外，直接访问可能会比较慢。我们可以使用 Cloudflare 提供的免费 CDN 服务来加速博客的访问速度。前往 Cloudflare 官网注册账号并添加你的域名，按照提示完成设置

安装完成后在文件夹中对空白区域右键应该有个git bash here如上图，点击后与注册号的GitHub关联之后应该是这样的

1. 查看所有 Git 配置

要查看包括系统级别在内的所有配置，可以使用：

git config --list

这将展示所有层面上的 Git 配置，包括全局、系统、和本地。

2. 查看特定配置项

如果你只想查看特定的配置项，如用户名，可以使用：

git config user.name

或者查看与远程仓库相关的 URL：

git config remote.origin.url

这些命令分别显示了用户名和当前仓库的默认远程仓库 URL。

3. 更改 Git 配置

如果你需要修改某项配置，例如改变用户名或邮箱，可以使用：

git config --global user.name "新用户名"
git config --global user.email "新邮箱地址"

与github关联后可以使用上面的指令查看是否成功，若看到账号邮箱与注册好的github账号关联成功，恭喜你，已经成功5%了！

舌尖上的二中（二）—— “面面”俱到

​ “快快快,别买早餐了要迟到了…”

​ 班上总有那么一两个同学,亦或家离学校太近,亦或冬日的被窝太温暖，总能看见他们在教室钟的最后通碟下,踩点气喘吁吁地来到教室,开始了一天的学习。清晨一二节课往往是学生“钓鱼执法”的最好时机，但对于这位早上差点迟到长大后却励志想当太空人的同学来说,星期六的早上,是他精神最好的时候，有着第二节课下课30分钟的吃饭时间，上课怎么能不兴奋起来呢?

​ 随着下课铃响，萋靡不振的空腹学生赴奔早餐战场,各家早餐店也在紧张准备着。“老板，大碗粉打包但在这里吃。”

​ “……”

​ 邵阳米粉不同于桂林米粉与其它米粉,它是由陈米与新米按照一定比例碾压制作出来的。口感较硬,不同于桂林米粉,桂林米粉丝滑柔顺，汤汁附着在上一带吃下。而那阳粉因其特殊的制作工艺,表面较粗糙，正宗的邵阳粉,每一根粉都能与每家特色的汤底融入其中,做到真正的入味。二中学姐刘乙如在参加《十三亿分见》时用邵阳话将邵阳米粉融入歌曲中：“脾酒哈打开;胃夜放肆呷,邵阳粉我爱,mylife is beauty. 木耳肉丝生炒牛肉生炒排骨大片牛肉酸辣肉末三鲜蛋饺还要加杂煎蛋.朝天椒,小米栋,剁辣椒干辣椒通通里面放！”

​ 豆腐木耳是各大粉店最畅销的系列了，豆腐吸收汤汁过后饱满而爽口，木耳与辣椒油混合一起,木耳便有了灵魂,清香清脆之中略带有一丝辣意。大片牛肉屈居第二,牛肉煮过的汤底和牛油,更让粉具有了神韵,吃牛肉粉,不在于牛肉，而在于粉，粉中夹杂着车油的味道。牛肉之中混着米香的味道，好不过瘾。酸辣肉丝是最近几年才大规模流行的，一碗真正的酸辣肉丝面,要能吃每一口面都有酸辣的味道，猪瘦肉配合着酸豆角,一唱一和一主一配，让人感受到来自面的味蕾冲击。

​ 细数二中周边各大粉店,因其各家店面,调味品,汤底,制法不同,相同的米粉在不同的店家不同的厨师的手里,绽放出不同的色彩与灵魂内涵。二中对面的苏记回民店最址人上瘾的,还是酸辣肉丝面,与北方不同，南方土地偏酸性且气候较温热，醒面过程中难免会有发酵变酸的现象，因此碱面成了不二之选，它的吃法也是最多的。眼镜粉面店因为面的独特口感以及菜品臊子的多样性,吸引了不同年龄段的顾客；老邵阳粉店开在十字路口,来来往住的人群和伏越地理区位以及时间的冲刷，让这家老字号店仍然屹立于此…

​ 南方碱面,北方宽面,西北手工面也是一道不可多得的美食，油泼面是灵魂,酸汤面是招牌，干拌面是精华。宽面的杰出代表就是油泼面，手工柔制，手工拉画，拉出来的面除面头以外宽窄粗细都差不多，以沸水烹饪,意制过程中将植物油放至电磁炉上温热,同时不断搅拌水中的宽面，待面煮至透明时，加以小青菜,约半分钟后出锅,晾干水分,装碗,加以葱蒜，辣椒,白菜,将温热好的油浇在上面。“嗞啦”一声，这是生命的碰撞,它们的声音就是灵魂的喜悦，面上跳动的辣椒粉便是生命的舞蹈，灵魂与生命交织,便构造出了一碗油泼面。在吃时也有讲究，先将面均匀搅拌过后可选择将油倒出来避免太腻，第一口下去,先是丝滑柔顺,与想象的粗糙大相径庭,再然后是葱蒜的油爆香,最后竟是辣椒的甜。一口下去,回味无穷。

​ 中国的美食，仅一个邵阳米粉与手工面便有十几种做法,南咸北甜,南碱北宽。可以说,每家店铺一点点调味的不同,便可造成舌尖上的味蕾上的感受千差万别,正因为这种差异,才造就了中华美食文化博大精深。

​ 吃粉前，我是邵阳的
​ 吃粉后，邵阳是我的

舌尖上的二中（一）—— 隔壁小孩都馋哭了
周记 2020/11/27

舌尖上的二中（一）—— 隔壁小孩都馋哭了

​ (作为一个资深吃货，我觉得我有必要为一些美食正名了)

​ 下午5:40，随着下课铃声响起，原本冷清的街道因为学生下课放学的到来而热闹起来，各家晚餐店，小吃店也跟着老板一声吆喝：“吃什么？”而忙碌起来。

​ 新奥尔良烤肉饭位于热闹与冷墙店铺的友界处。但这依旧阻止不了学生对它的喜爱。店主张大妈自开店以来一直兼承质量第一，口感第一为原则，这也勾起了毕业生坐在他乡的思念”每每回到二中总要来一口烤肉饭，吃完便说：”还是当年的味道。”

​ “大碗招牌烤肉拌饭。” “这里……” 与其它店面不同，这里上菜的却是店主张大妈两个双胞胎女儿。在等待期间，最大的亮点便是张大妈在靠近就树的江边的地方修膳了一些座位。可供食客在临近夕阳西下时观看夕阳与邵水河水天一色的独特视觉美景。

​ 张大妈为了使米饭具有更好的口感，通常提前煮好饭而温热，这样，食客在吃时不致于感到太烫而无从下口， 赶时间的学生也能尽快吃完饭。李大妈的烤肉饭不同于白米饭，在制作烤肉时，将剩下的汤汁与白米饭充分混合，吸收，手工搅拌。出来的半成品米饭届时便具有了一丝神韵，它既保留了南方大米特有的米香，又混入了让人食欲大开的肉香。再将烤肉手撕下来均匀混合在米饭之中，看似无肉，一口下去实则满嘴是肉，香而不赋。配以土豆丝，胡萝人丝调味，生菜配相。番茄沙拉平分秋色，黑胡椒照烧此各有所爱，招牌孜然奇特经典，还有免费的紫菜蛋汤供狼吞虎咽或是细嚼慢咽的食客开胃，解腻，顺肠——烤肉拌饭是情感的升温，紫菜蛋汤是情感的蜜意。

​ 相比于李大妈的四年老店，小巷里新开的脆皮鸡类饭是一家后起之秀的店，刚一开张不到一周便悉数好评。店主张大哥李大哥是两位三十多岁的年轻人，因为年轻，思想紧跟潮流，店内装饰与色调更符合年轻人的口味，店外的节日小彩灯与店内暖色调朝相辉映，给人在寒冷的冬夜一种温暖的感觉。最让人眼前一亮的还是张大哥免费提供的汤水一以大块排骨熬汤，将萝卜，小块瘦肉放入高汤之中，辅以葱，姜为佐，悉数慢炖，提前两小时将高汤温热，待食客来时温度刚刚好，不烫不冷，萝卜入口即化，甜而丝滑有余味，汤中有骨头的精华，生姜调味，小葱配相，让人在寒冷的冬夜里在入店即可感受到美食的温暖。

​ 李大哥店内独特的饭前招特更是这家店的成功秘诀之一，桌旁的开胃菜如腌黄瓜，萝卜条之类的让食客在等待之余不略显单调，免费的萝卜汤将店主对美食的理解与对食客的尊重携着香气飘扬在冬夜的街道上，播洒在人们的心头，这家店印证了“酒香不怕巷子案”的俗语。

​ 无论是张大妈还是李大哥，或许正是因为他们这种精神，对美食精益求精的态度，了解食客真正所需。我们高中生活为时不长的体息时间才不单调，或许多年以后重走二中，看到店铺依旧趟立在那儿，我们会说:”看，那家店我读书时经常吃，老板都认得我呢!(本文人名纯属虚构，但故事情节真真切切实实)
